Microsoft Word (MS Word) is a powerful word-processing application developed by Microsoft. It enables users to create, edit, format, save, and print a wide variety of documents. With tools like spell check, grammar correction, and a built-in thesaurus, it helps improve writing accuracy. Users can insert tables, images, charts, and headers or footers to make documents more structured and visually appealing. Advanced features include mail merge for personalized bulk letters, track changes for collaboration, and real-time co-authoring. Word integrates seamlessly with other Microsoft Office apps and supports cloud saving for easy access.

Microsoft Excel is a powerful spreadsheet software developed by Microsoft. It organizes data in a grid of rows and columns, allowing users to enter numbers, text, or dates into individual cells. Excel supports hundreds of built-in functions and formulas — like SUM, IF, and VLOOKUP — to perform complex calculations and automate tasks. It also offers data analysis tools such as PivotTables, what-if analysis, and scenario manager. Users can visualize data through charts and graphs, and automate repetitive work using macros via Visual Basic for Applications (VBA).
